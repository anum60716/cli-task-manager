# 🚀 Elite CLI Task Manager

A powerful and simple Command Line Task Manager built with **Python**, **Typer**, and **SQLite**.  
Manage your daily tasks directly from the terminal with speed and efficiency.

---

## 📌 Features

✅ Add new tasks  
📋 View pending/completed/all tasks  
✔ Mark tasks as completed  
🗑 Delete tasks  
🔍 Search tasks by keyword  
📊 View task statistics  
💾 Persistent storage using SQLite database  

---

## 🛠 Technologies Used

- Python 3
- Typer (CLI framework)
- SQLite3
- Pathlib
- Datetime
